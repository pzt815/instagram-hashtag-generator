# 📸 Instagram Hashtag Generator

An open-source AI tool that automatically generates 
SEO-optimized Instagram hashtags using OpenAI Codex API.

## 🚀 Features
- Auto-generate 30 optimized hashtags by category
- Supports fashion, beauty, food, interior, luxury niches
- OpenAI Codex-powered natural language processing
- Simple CLI & Python interface

## 🛠 Tech Stack
- Python 3.10+
- OpenAI Codex API
- Instagram Graph API

## 📦 Installation
pip install -r requirements.txt

## 💡 Usage
python hashtag_gen.py --category "fashion" --lang "ko"

## 🌍 Use Case
Built for Korean SNS content creators and e-commerce 
operators managing multi-category Instagram channels.
